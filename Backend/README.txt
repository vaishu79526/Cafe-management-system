
# Spring Boot Backend
This folder contains the code for the Spring Boot backend of the Cafe Management System.

## Instructions
1. Open the project in an IDE like IntelliJ IDEA or Spring Tool Suite (STS).
2. Add the `src` folder to your IDE.
3. Configure your `application.properties` file to match your MySQL database settings.
4. Run the application using the main class `CafeManagementApplication.java`.

## Features
- REST APIs for managing users and menu items.
